A plugin which will activate a shortcode that allows an admin user to quickly add or remove static pages.


== Installation ==

1. After downloading the plugin, install it using the standard WordPress plugin installation. Access PLUGINS/ADD NEW and upload the plugin you want to install.
2. Place the shortcode 'padder_add_page' on the page you want to have the functionality enabled.
